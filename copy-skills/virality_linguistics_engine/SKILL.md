# Virality Linguistics Engine

Linguistic triggers of virality and shareability.

## When to use

- Final copy polish
- Shareability optimization
- Memorable phrase crafting
- Engagement language

## Patterns

- **Contrast language**: before/after, us/them, was/now
- **Moral language**: right, wrong, should, must
- **Urgency language**: now, before, last chance
- **Identity language**: we, our, together

## Rules

| Rule | Value |
|------|-------|
| Prefer short sentences | true |
| Prefer concrete words | true |
| Avoid complex jargon | true |
