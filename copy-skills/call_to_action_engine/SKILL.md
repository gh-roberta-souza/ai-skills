# Call to Action Engine

Action triggers for campaign completion.

## When to use

- Campaign captions
- Advocacy content
- Petition drives
- Awareness posts

## CTA types

- **Awareness**: share, follow, learn more
- **Advocacy**: sign, contact, demand
- **Petition**: add name, support, join

## Placement

| Location | Use case |
|----------|----------|
| End of caption | Primary CTA |
| Mid-caption | Secondary action |
| First line | Urgent/scroll-stopping |

## Integration

Invoked by `copy_ai_orchestrator` during Stage 7 (distribution layer).
