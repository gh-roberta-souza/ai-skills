# Information Density Optimizer

Controls cognitive load for maximum comprehension.

## When to use

- Dense content simplification
- Carousel text optimization
- Caption length decisions
- Multi-slide content distribution

## Rules

| Rule | Value |
|------|-------|
| Max words per sentence | 16 |
| Max sentences per caption | 6 |

## Principles

- Reduce redundancy
- Prioritize key information
- One claim per sentence when possible
