# Narrative Structuring Engine

Controls storytelling flow and narrative arc.

## When to use

- Investigative journalism style
- Awareness campaigns
- Impact storytelling
- Carousel narratives

## Story models

### Investigative sequence

1. Hook
2. Problem
3. Evidence
4. Escalation
5. Accountability
6. Resolution

### Awareness sequence

1. Hook
2. Explanation
3. Impact
4. Action
