# Persuasion Frameworks Engine

Proven persuasion frameworks for editorial and marketing copy.

## When to use

- Structuring long-form content
- Campaign copy development
- Carousel narrative flow
- Advocacy and awareness content

## Frameworks

### AIDA

| Stage | Purpose |
|-------|---------|
| Attention | Capture initial interest |
| Interest | Build relevance |
| Desire | Create want or need |
| Action | Drive response |

### PAS

| Stage | Purpose |
|-------|---------|
| Problem | Identify pain point |
| Agitation | Amplify stakes |
| Solution | Present resolution |

### BAB

| Stage | Purpose |
|-------|---------|
| Before | Current state |
| After | Desired state |
| Bridge | Path between |

## Selection rules

| Content type | Framework |
|--------------|-----------|
| Investigative content | PAS |
| Activism campaign | AIDA |
| Awareness posts | BAB |

## Integration

Invoked by `copy_ai_orchestrator` during Stage 3 (persuasion structure).
