# Curiosity Gap Generator

Creates scroll-stopping curiosity and sustained engagement.

## When to use

- Carousel content (swipe-through)
- Long-form captions
- Thread-style content
- Email subject lines

## Methods

- **Incomplete information**: hint without full reveal
- **Contrast reveal**: set up tension before resolution
- **Question hook**: open loop requiring answer
- **Unexpected statistic**: surprising number demanding context

## Rules

| Rule | Value |
|-----|-------|
| Never reveal full story in first sentence | true |
