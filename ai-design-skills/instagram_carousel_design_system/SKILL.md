# Instagram Carousel Design System

Multi-slide layout system for narrative continuity and swipe-driven engagement.

## When to use

- Carousel posts (5–10 slides optimal)
- Educational threads
- Multi-step storytelling
- Content requiring narrative progression

## Slide roles

| Role        | Purpose                    |
|-------------|----------------------------|
| hook        | Capture attention          |
| problem     | Establish context          |
| explanation | Deliver content            |
| insight     | Key takeaway               |
| summary     | Recap                      |
| call_to_action | Drive action           |

## Cross-slide rules

- **Grid consistency**: true
- **Typography consistency**: true
- **Color palette locked**: true

## Narrative engine (progression model)

1. **Stage 1 — Curiosity**: Hook slide, low information
2. **Stage 2 — Context**: Problem/explanation
3. **Stage 3 — Insight**: Peak information
4. **Stage 4 — Payoff**: Summary
5. **Stage 5 — Action**: CTA

## Swipe psychology

- **Curiosity gap**: enabled
- **Information release curve**: start low → middle peak → end resolution

## Visual progression

- **Hook slide**: largest element size
- **Density curve**: slide 1 medium → slides 2–4 high → final slide medium

## Continuity constraints (element position lock)

- Logo: fixed
- Footer: fixed
- QR code: fixed

## Integration

Uses `instagram_design_system` for per-slide layout; applies carousel-specific narrative and consistency rules.
