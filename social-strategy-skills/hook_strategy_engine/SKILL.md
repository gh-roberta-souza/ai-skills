# Hook Strategy Engine

Defines the campaign's core attention hook.

## When to use

- Campaign hook design
- First-slide strategy
- Headline definition
- Attention capture across series

## Hook models

- **Shocking statistic**: quantified impact
- **Moral statement**: rights, wrongs, should
- **Contradiction**: tension between expectation and reality
- **Urgency signal**: time-sensitive stakes

## Rules

| Rule | Value |
|------|-------|
| Hook must be clear in 2 seconds | true |

## Example

```
71 complaints.
0 solutions.
```

## Integration

Invoked by `social_strategy_orchestrator` during Stage 3 (hook design).
