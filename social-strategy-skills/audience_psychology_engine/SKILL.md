# Audience Psychology Engine

Behavioral science for message resonance.

## When to use

- Strategy refinement
- Message optimization
- Emotional resonance design

## Psychology models

- **Loss aversion**: what is at risk if nothing changes
- **Injustice perception**: unfairness framing
- **Moral outrage**: right vs wrong activation
- **Social proof**: others care, others act
