# Social Graph Intelligence

Maps relationships and influence networks from open-source social data.

## When to use

- Mapping relationship networks
- Identifying key influencers
- Detecting clusters and communities
- Analyzing information flow

## Node types

- User accounts
- Organizations
- Communities

## Relationship types

- Follower relationship
- Interaction (likes, comments, shares)
- Collaboration
- Shared affiliation

## Network metrics

- **Degree centrality**: connection count
- **Betweenness centrality**: bridge nodes
- **Clustering coefficient**: community density

## Analysis goals

- Identify key influencers
- Detect clusters
- Map information flow

## Integration

Invoked by `osint_ai_orchestrator` during Stage 2 (data collection).
