# Layout AI Orchestrator

Master orchestration system coordinating all design skills for optimized Instagram layouts.

## When to use

- Creating any Instagram content
- Multi-step layout generation
- Content requiring layout optimization
- Carousels, infographics, or single posts

## Purpose

- Coordinate design skills
- Optimize layout generation
- Ensure global consistency

## Managed skills

1. instagram_design_system
2. instagram_carousel_design_system
3. viral_content_layout_patterns
4. data_visualization_for_instagram
5. psychology_of_attention_layout
6. visual_balance_solver
7. layout_entropy_control
8. virality_prediction_engine
9. algorithmic_typography_system

## Pipeline stages

### Stage 1: Input analysis

- Detect content type
- Detect layout goal
- Detect visual priority

**Content types**: single_post, carousel, infographic, data_visualization, marketing_post

### Stage 2: Skill selection

| Condition | Activate |
|-----------|----------|
| Single post | instagram_design_system, psychology_of_attention_layout, algorithmic_typography_system |
| Carousel | instagram_carousel_design_system, instagram_design_system, psychology_of_attention_layout |
| Data visualization | data_visualization_for_instagram, instagram_design_system |
| High engagement goal | viral_content_layout_patterns, virality_prediction_engine |

### Stage 3: Layout generation

1. Generate base grid
2. Generate typography structure
3. Place primary focus
4. Distribute secondary elements

**Constraints**: grid alignment required, baseline alignment required

### Stage 4: Layout optimization

**Modules**: visual_balance_solver, layout_entropy_control, attention_heatmap_alignment

**Goals**: maximize readability, maximize visual balance, minimize clutter, maximize attention capture

### Stage 5: Virality evaluation

**Engine**: virality_prediction_engine

| Threshold | Interpretation |
|-----------|----------------|
| 60–75    | Acceptable |
| 75+      | High performance |

**If score low**: increase contrast, simplify layout, increase hero prominence

### Stage 6: Validation

**Checks**: grid alignment, typography consistency, density balance, visual balance, entropy range, no overlap

**Fail conditions**: misaligned grid, unreadable text, chaotic layout, excessive whitespace

## Global constraints

- Max colors per design: 4
- Max typography levels: 4
- Density range: 0.60–0.85

## Optimization strategy (multi-objective)

| Objective | Weight |
|-----------|--------|
| Readability | 0.35 |
| Attention | 0.25 |
| Visual balance | 0.20 |
| Virality | 0.20 |

## Output requirements

- Clear visual hierarchy
- Consistent grid alignment
- Balanced visual weight
- Mobile readability
- Predictable attention flow

## Failure handling

- Retry pipeline: true
- Max iterations: 3

## Learning feedback (optional)

- Store high-virality layout patterns
- Improve future layout generation
