# Psychology of Attention Layout

Layout optimization using cognitive psychology, Gestalt principles, and visual salience.

## When to use

- Optimizing attention flow
- Improving readability
- Establishing perceptual hierarchy
- Reducing cognitive load

## Gestalt principles

| Principle   | Application                    |
|-------------|--------------------------------|
| Proximity   | Group related elements         |
| Similarity  | Consistent styles across groups|
| Continuity  | Directional guidance           |
| Figure-ground | Strong contrast             |

## Cognitive rules

- **Miller's Law**: max 7 visual groups
- **Chunking**: enabled — group information into digestible units

## Attention zones

| Zone           | Location      | Priority |
|----------------|---------------|----------|
| Primary focus  | Upper center  | Highest  |
| Secondary focus| Center left   | High     |
| Action zone    | Bottom right  | CTA      |

## Reading patterns (mobile eye-tracking)

- F-pattern
- Z-pattern
- Center bias

## Visual salience factors

- Size
- Contrast
- Color
- Position
