# Viral Content Layout Patterns

Layout patterns optimized for social virality using psychological triggers and rapid comprehension.

## When to use

- High-engagement goals
- Shareable content
- Rapid comprehension required
- Emotional or curiosity-driven posts

## High-engagement structures

### Big number pattern

- **Layout**: hero number + short caption + icon support
- **Use case**: statistics, milestones, data highlights

### Bold statement pattern

- **Layout**: bold claim + high-contrast background + minimal text
- **Use case**: opinions, provocations, identity alignment

### Curiosity gap pattern

- **Layout**: incomplete statement + hint visual + continuation on next slide
- **Use case**: carousels, teasers, "you won't believe..."

## Engagement rules

- **Cognitive load**: max 30 words per slide
- **Hero size ratio**: 1.8× (hero element dominates)

## Emotional triggers

- Surprise
- Usefulness
- Novelty
- Identity alignment

## Integration

Activated by `layout_ai_orchestrator` when `high_engagement_goal` is detected; pairs with `virality_prediction_engine` for scoring.
